 Ниже приведён пример каркаса программы на Python с использованием Tkinter, соответствующий вашему описанию ВКР:

Пользователь выбирает/вводит:
Сегменты сети
Подсети
Сервисы/порты
Матрица взаимодействия (правила)
Количество сетевого оборудования
Программа:
Валидация модели (проверка пересечений подсетей, дублирования/конфликта правил)
Создаёт отчёт и визуализацию
Позволяет сохранить результат в файл

Структура проекта

network_segmentation_tool/
│
├── main.py                # Основной GUI и логика
├── validation.py          # Функции валидации
├── report_generator.py    # Генерация текстового отчёта
├── visualizer.py          # Визуализация (например, через matplotlib)
└── example_data.py        # Заранее заданные варианты для выбора
